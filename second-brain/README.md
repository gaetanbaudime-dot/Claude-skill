## Utilisation

1. Ouvrir ce dossier comme vault dans Obsidian (Open folder as vault).
2. Graph view (Ctrl+G) : les couleurs par type sont préconfigurées dans `.obsidian/graph.json` (concept bleu, personne vert, expérience orange, controverse rouge, théorie indigo, débat violet, méthode cyan, école jaune, terme gris).
3. Naviguer depuis [[_MOC]], ou chercher un biais par son nom français.

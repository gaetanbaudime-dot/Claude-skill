# Journal de coaching

> [!info] Sessions de coaching avec Claude
> Chaque session de coaching (décision difficile, revue de stratégie, remise en question) laisse une trace ici : question posée, options considérées, recommandation, décision prise. La relecture périodique de ce journal est le meilleur outil de calibration qui existe : comparer ce qu'on prévoyait à ce qui est arrivé attaque directement l'[[Excès de confiance]] et le [[Biais rétrospectif]].

## Format de session

### AAAA-MM-JJ : [Sujet]
- **Question / enjeu** :
- **Options considérées** :
- **Recommandation de Claude** (et pourquoi) :
- **Décision prise** :
- **Prédiction** (qu'est-ce qui doit se passer, d'ici quand, pour dire que c'était le bon choix) :
- **Revue a posteriori** (à remplir à l'échéance) :

## Règles du coaching

1. Claude joue l'avocat du diable par défaut : il cherche ce qui casse le plan avant ce qui le confirme (voir [[Débiaisage]], technique "considérer l'opposé").
2. Toute décision importante passe par un [[Débiaisage|pré-mortem]] : imaginer l'échec dans 12 mois et raconter pourquoi.
3. Les prédictions sont écrites AVANT l'issue, jamais après.
